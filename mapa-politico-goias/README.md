# Mapa Político Goiás (WordPress)

Plugin WordPress com cadastro **manual** de políticos, apoio controlado de IA para texto e exibição em mapa com Leaflet + OpenStreetMap.

## Fluxo atual

- Cadastro manual no wp-admin
- Upload manual de foto via Media Library
- IA apenas para preencher:
  - Biografia
  - Histórico político
- Sem sincronização automática
- Sem filas
- Sem cron de IA

## Shortcode

Use:

```txt
[mapa_politico]
```

ou

```txt
[mapa_politico_goias]
```

## Menus

- Cadastro Manual
- Logs da IA
- Excluir Cadastros

## Requisitos

- WordPress 6+
- PHP 8+
